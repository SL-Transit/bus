# แอพส่ง GPS รถโดยสาร

แอพนี้ติดตั้งบนมือถือคนขับแต่ละคัน เพื่อส่งตำแหน่งรถเข้า Firebase Realtime Database

## การใช้งาน

1. ติดตั้ง APK บนมือถือ Android
2. เปิดแอพ `ส่ง GPS รถ`
3. อนุญาตตำแหน่งเมื่อมือถือถาม
4. แอพจะเริ่มส่ง GPS ไป `bus/car1` อัตโนมัติ

แอพจะส่งข้อมูลไปที่ `bus/car1` ทุก 20 วินาที

## หมายเหตุ

- แอพทำงานเป็น foreground service จึงมี notification ค้างไว้ขณะส่ง GPS
- คนขับไม่ต้องเปิดหน้าแอพค้างไว้ แต่ไม่ควรปัด notification หรือ force stop แอพ
- หากมือถือเปิดโหมดประหยัดแบตมากเกินไป อาจต้องอนุญาตให้แอพทำงานเบื้องหลัง
